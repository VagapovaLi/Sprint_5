# Проект автоматизации тестирования онлайн заказа бургеров.
## Описание
# Этот проект представляет собой веб-приложение для онлайн-заказа бургеров. Пользователи могут создать свой заказ, выбрав различные виды бургеров,соусов и начинок.
## Функциональность
- Просмотр меню с различными бургером и добавками
- Возможность выбора ингредиентов для создания индивидуального бургера
- Оформление заказа и выбор способа оплаты
